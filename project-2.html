Project Overview

Here, we used a kaggle dataset to predict the Breast Cancer. People will be giving the values of: "radius_mean texture_mean perimeter_mean area_mean smoothness_mean compactness_mean concavity_mean concave points_mean symmetry_mean fractal_dimension_mean ... fractal_dimension_se radius_worst texture_worst perimeter_worst area_worst smoothness_worst compactness_worst concavity_worst concave points_worst symmetry_worst" to get the results. Also graphical representation present to make it more clear.
Basically here we imported the data firstly. Then we done the data cleaning process by dropping unwanted cols. Then did the label encoding on the specific cols where we got string vals. Then separated the feature and targeted data. Then after training and testing the data we done normalization and standard scaling. Then plotted some graphs based on those data. Then used SVM, Decision Tree and Random Forest Algorithms. From these we found that random forest gives the best accuracy reports.

Flexible to use.
